## 📖 如何使用这个笔记库

1. **查询时**：按索引表点进对应主题，里面都按「概念 → 命令/代码 → 坑 / 常见问题」组织
2. **记录新内容时**：在对应主题文件末尾追加，或新建 `06_xxx.md` 然后更新本 README
3. **遗忘时**：先看每篇开头的"核心要点 3 句话"复习
4. **答疑时**：每篇底部都有"踩坑记录"，可以把自己遇到的新坑补充进去

---

## 🔖 命名约定

- 文件名前缀 `NN_`：两位数字便于排序，比如 `01_`、`02_`
- 主题名用**中文**：这是个人学习笔记，中文比英文更自然好查
- 标题层级：`##` 是一级主题，`###` 是二级主题，避免超过 4 层嵌套
